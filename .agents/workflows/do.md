# Run Task Workflow

---
description: docs/REVISIONS.md 内のタスクID（例: R1）を指定して作業を自動開始する
---

## 1. タスクの特定
1. **REVISIONS.md の確認**
    - `docs/REVISIONS.md` を開き、指定された ID（例: `[R1]`）に該当するタスク内容を読み取る。
2. **作業対象の確定**
    - 読み取ったタスクの内容を「現在のリクエスト」としてセットする。

## 2. 作業の開始
1. **Editワークフローへの委譲**
    - 特定したタスク内容に基づき、`@[/edit]` ワークフローを起動する。
    - 以降は `/edit` の手順に従い、ブランチ作成から実装プランの作成へと進む。
