# LogiDo 実装計画書 (Implementation Plan)

## フェーズ 1: プロジェクト基盤とデザインシステム
- **環境構築**: Vite + React による高速な開発環境のセットアップ。
- **CSS変数定義**: `ui_ux_design_spec.md` に基づくカラー、タイポグラフィ、スペーシングの定義。
- **レイアウト実装**: アプリ全体のシェル（枠組み）を作成。

## フェーズ 2: データ・ロジック層
- **ツリー管理エンジン**: ノードの追加・削除・移動（再ペアレンティング）を処理するロジック。
- **進捗計算エンジン**: `progress_sync_spec.md` に基づく再帰的なロールアップ計算。
- **ストレージ層**: LocalStorage を用いたデータの永続化。

## フェーズ 3: List View (実行フェーズ)
- **階層リストコンポーネント**: 再帰的なレンダリングによるインデントリストの表示。
- **インタラクション**: ステータス変更、インライン編集、キーボード操作。

## フェーズ 4: Why/How インスペクター
- **コンテキストパネル**: 選択ノードのパス表示と子ノード一覧。
- **ロジカル・サジェスト**: 「分解する」ボタン等の思考支援UI。

## フェーズ 5: Tree View (視覚的戦略フェーズ)
- **SVG レンダリング**: D3.js などのライブラリ、または自作の配置アルゴリズムによる描画。
- **キャンバス操作**: ズーム、パン、ノードの展開/折りたたみ。

## フェーズ 6: ポリッシュと高度な機能
- **ドラッグ＆ドロップ**: 論理構造の組み換え UI。
- **アニメーション**: パーティクル効果、スムーズなビュー遷移。
- **MECE チェック**: 重複や漏れの警告表示ロジックの実装。
