# 実装報告：一直線フロー表示 (Linear Flow View)

## 1. 概要
TreeView において、従来の階層的なツリー表示と、実行順序に基づいたボトムアップの一直線フロー表示（Flow モード）を切り替える機能を実装しました。

## 2. 実装済み内容

### 2.1 データロジック (`treeLogic.js`)
- `getFlattenedFlow` 関数を実装。
- ツリーを **後行順（Post-order）** で走査し、アクションから目標へと向かう配列を生成。

### 2.2 UI・レンダリング (`TreeView.jsx`)
- `layoutMode` (Tree/Flow) および `flowOrientation` (Horizontal/Vertical) の状態管理を導入。
- D3.js による動的な座標計算と、モード切り替え時のアニメーションを実装。
- **垂直レイアウト**: 目標が最下部になるように Y軸座標を割り当て。

### 2.3 スタイル (`TreeView.css`)
- 流れるようなアニメーションを持つ `flow-link` スタイル。
- ガラス質感のコントロールパネルとモード切り替えボタン。
- ホバー時のハイライト機能（`!important` を用いた確実な表示）。

## 3. 完了したテスト項目
- 水平/垂直の切り替えが正しく動作すること。
- 目標（Goal）が正しくフローの最後に配置されていること。
- ホバー時に接続線が消えず、正しく強調されること。
