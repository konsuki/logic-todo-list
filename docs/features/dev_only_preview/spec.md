# 仕様書: 開発者限定プレビュー機能 (Developer-Only Preview)

## 1. 目的
デザインプレビュー機能（DesignSandbox）が、一般ユーザーの利用する本番環境（プロダクションビルド）では表示・実行されないように制限し、機密性と安全性を確保する。

## 2. 制限対象
以下の機能を「開発モード（npm run dev）」実行時のみに限定する。
- **UIコンポーネント**: ヘッダー右側に表示される「稲妻アイコン（プレビュー切り替えボタン）」。
- **キーボードショートカット**: `Alt + P` (Mac: `Option + P`) による表示モードの切り替え。

## 3. 技術的実現方法
Viteが提供する環境変数 `import.meta.env.DEV` を使用して判定を行う。

- `import.meta.env.DEV` が `true` の場合:
    - ヘッダーにアイコンを表示する。
    - キーボードショートカットを登録・実行する。
- `import.meta.env.DEV` が `false` の場合:
    - アイコンをレンダリングしない。
    - ショートカットが押されても何もしない（またはイベントリスナーを登録しない）。

## 4. 影響範囲
- `src/App.jsx`: ヘッダーのレンダリング条件。
- `src/hooks/useShortcuts.js`: ショートカットの実行条件。
