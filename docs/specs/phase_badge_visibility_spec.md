# 機能仕様：フェーズバッジ（実行フェーズ）の表示・非表示設定

## 1. 目的
リストビューにおいて、各タスクに表示されている「準備期」「実行期」などのフェーズバッジの表示・非表示を切り替えられるようにし、ユーザーが好みに応じてリストの密度を調整できるようにする。

## 2. 変更内容

### 2.1 設定項目
- 設定名: `showPhaseBadges`
- 型: `boolean`
- デフォルト値: `true`

### 2.2 UI/UX
- **設定パネル**: 「表示設定」セクションに、「フェーズバッジを表示」のトグルスイッチを追加する。
- **リストビュー**: `showPhaseBadges` が `false` の場合、タスク名の横または下に表示されているフェーズバッジ（`.phase-badge`）を非表示にする。

### 2.3 多言語対応 (i18n)
- `settings.show_phase_badges`: 「フェーズバッジを表示」 / "Show Phase Badges"
- `settings.show_phase_badges_desc`: 「リストビューで各タスクの実行フェーズを表示するかどうかを切り替えます。」 / "Toggle visibility of execution phase badges in the list view."

## 3. 考慮事項（より便利にするために）
- **プレビュー機能**: 設定を変更した際、即座にリストビューに反映されるようにする（既存の `SettingsContext` の仕組みで対応可能）。
- **デフォルト値**: 既存のユーザー体験を損なわないよう、デフォルトは `true`（表示）とする。
