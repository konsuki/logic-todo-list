## 🚀 リクエストの書き方（例）

- **[ステータス]**: `[TODO]` / `[PLANNING]` / `[IN_PROGRESS]` / `[DONE]`
- **[内容]**: 具体的な変更内容、現状の課題、期待する結果
- **[日付]**: 2026-04-28
